💻 CLI Usage Quick reference for connecting the Tela CLI to this portal.

After generating a token above, register this portal as a remote in the Tela CLI:

tela remote add awansaya https://awansaya.net

When prompted for a token, paste the value you copied above. The CLI will verify the connection and store the token securely.

To check your configured remotes:

tela remote list
